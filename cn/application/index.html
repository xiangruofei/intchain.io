项目介绍 当理想照耀现实，我们才会发现世界如此美好

INT是全球首个从物联网结构体系演进的角度，自下而上打造出的新一代物联网区块链系统。
“她”将为各类分布式物联网应用及产业生态提供一个数据来源可靠、传输与存储安全、保密且无需信任的基础设施，并且塑造一种以设备为主体的新商业模式，开辟全新市场空间。

项目亮点 采用业界领先的物联网层次结构模型

物联网涉及复杂的通信技术，不同于计算机网络所采用OSI参考模型的二维结构，通信领域的网络层次模型是三维结构，因此必须充分考虑到隔层通信的问题，应对感知与执行节点能量受限，尽可能简化节点结构。
INT采用最新的物联网层次结构模型，既考虑了到通信领域的跨层设计，也增加了信息安全、网络管理等互联网的共性服务。

拥有更加强大的共识机制

从工作量证明PoW机制到权益证明PoS机制，再到EOS团队采用的DPoS机制，以及最近备受关注的基于Algorand算法的类PoS机制，INT chain根据物联网分布式账本的特征演化，将不断优化匹配共识机制。

成为物联网体系结构最新研究成果的应用典范

体系结构是对复杂系统的一种抽象，也是系统建模的一种方法。
INT整合了物联网体系结构研究的最新成果，可以针对不同物联网应用场景抽象出相应的体系结构，再甄选出共性特征，进行归纳。

首推开放的物联网公链，人人可触摸

INT chain对所有用户无门槛公开，可匿名参与，不设网络和区块链访问权限。
INT chain将利用包括密码哈希函数和椭圆曲线公钥密码技术的现代密码学技术，保证交互数据的不可篡改，验证识别节点及确定经济权益等。

技术优势

目前最合理的区块链-物联网云架构

重新修正传统云计算为中心的物联网体系结构，融入了雾/边缘计算和SDN技术，对今后几年的物联网演进趋势的深刻理解，抽象出了INT定义的分布式区块链IoT架构

安全可靠的区块链网关设计

用INT Blockchain Trust Management（信任管理系统） 替代传统无安全保障的类边界网关协议（BGP， Border Gateway Protocol）和中心化的公钥基础设施PKI等域间路由协议

公链架构的分离优化

依据INT重新定义的区块链IoT架构，创造性地将INTchain作了INT kernel 和 INT shell的架构分离，既可以保证适应IoT应用场景的灵活性（高可用和高可扩展性），又可以保证区块链的一致性和分区容错性

改进的共识机制

INT kernel将部署改进的Algorand共识算法（以Byzantine agreement protocol为核心）应对复杂物联网场景下可能带来的女巫攻击（“Sybil Attack”）和拒绝服务攻击（”DoS Attack”）
