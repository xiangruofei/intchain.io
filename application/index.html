Project Introduction When the ideal shines reality, we will find the world so beautiful

From the perspective of the evolution of the IoT structure, INT is the world's first new generation of bottom-up created system of IoT and blockchain. INT will provide series of decentralized IoT applications and industrial ecology with a reliable data source, transmission and storage security, confidential and untrusted infrastructure and shapes a new business model focused on devices, which opens up a new market.

Project Highlights The industry leading IoT hierarchical structure model

IoT involves complex communication technologies, which is different from the two—dimensional structure of the OSI reference model adopted by computer networks. The network level model in the field of communication is three-dimensional. Therefore, we should fully consider the problems of interlayer communication and simplify the node structrue as much as possible to solve the problems of sensing and implementing nodes with limited energy.
INT adopts latest IoT hierarchical structure model, which not only considers cross-layer design of telecommunication but also improves common service of Internet, such as information security, network management and so on.

More Powerful Consensus Mechanism

From PoW mechanism、PoS mechanism、 EOS team’s DPoS mechanism to the recent high-profile PoS mechanism based on Algorand algorithm. INT chain constantly optimizes and quests for the most suitable consensus mechanism according to the evolution characteristics of the IoT distributed ledger.

The application examples of the latest research results of IoT architecture

The system architecture is an abstraction of complex systems and a system modeling method.
INT integrates the research achievement of the Internet of things architecture, which can abstract the corresponding architecture for different IoT application scenarios, screen the commonalities and then summarize them.

An open public IoT chain for everyone

INTChain is open to all users without requirements, network and blockchain access permission. It can ensure the irreversibility of the interactive data, verify the identification node, determine the economic rights and interests by taking advantage of cryptographic hash function and elliptic curve public key cryptological technologies. The users can anonymously participate at ease.

Technical Advantage

The most logical blockchain-IoT cloud architecture

INT amends the traditional cloud computing-centric IoT architecture, integrates fog and edge computing as well as SDN technology. We grasp the trend of the evolution of the Internet of things in the next few years to abstract out the INT defined distributed blockchain IoT architecture.

The secure and reliable blockchain gateway design

INT adopts INT Blockchain Trust Management instead of Inter-domain routing protocol, such as the traditional unprotected similar Border Gateway Protocol (BGP) and centralized Public Key Infrastructure (PKI)

The separated and optimized public chain architecture

On the basis of INT redefine blockchain IoT architecture, INT creatively separates INTchain into INT kernel and INT shell framework, which not only ensures the flexibility (high availability and scalability) while adapting to IoT application scenarios, but also ensures the consistency and partition tolerance of blockchain technology.

The improved consensus mechanism

The INT kernel will deploy the improved Algorand consensus algorithm (centered on the Byzantine agreement protocol) to deal with possible "Sybil Attack" and "DoS Attack" in complex IoT scenarios.
